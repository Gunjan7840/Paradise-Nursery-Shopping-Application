# Paradise-Nursery-Shopping-Application
🌿 Paradise Nursery Shopping Application

Paradise Nursery Shopping Application is a React-based web application that provides an online platform for purchasing nursery plants. The application allows users to explore different plants, view their details, and add selected items to a shopping cart for a smooth and convenient shopping experience.

✨ Features

Browse a variety of nursery plants

View plant details like name and price

Add and remove plants from the cart

Simple and user-friendly interface

🛠️ Technologies Used

React.js

JavaScript (JSX)

CSS

Git & GitHub

▶️ How to Run
npm install
npm start
